# પહેલું પહેલું મંગળિયું વરતાય રે

પહેલું પહેલું મંગળિયું વરતાય રે,</br>
પહેલે મંગળ ગાયોનાં દાન દેવાય રે,</br>
અગ્નિદેવની સાક્ષીએ ફેરા ફરાય રે,</br>
સૌને હૈયે આનંદ અતિ ઉભરાય રે.

બીજું બીજું મંગળિયું વરતાય રે,</br>
બીજે મંગળ રૂપાંનાં દાન દેવાય રે,</br>
માંડવડામાં મંગળગીતો ગવાય રે,</br>
શુભદિન આજે શુકનનો કહેવાય રે.

અગ્નિદેવીની સાક્ષીએ ફેરા ફરાય રે</br>
સૌને હૈયે આનંદ અતિ ઊભરાય રે

ત્રીજું ત્રીજું મંગળિયું વરતાય રે,</br>
ત્રીજે મંગળ સોનાંનાં દાન દેવાય રે,</br>
ફૂલડાં કેરી ફોરમ બધે પ્રસરાય રે,</br>
બન્ને પક્ષે આનંદ અતિ ઉભરાય રે.

ચોથું ચોથું મંગળિયું વરતાય રે,</br>
ચોથે મંગળ કન્યાનાં દાન દેવાય રે,</br>
અગ્નિદેવીની સાક્ષીએ ફેરા ફરાય રે,</br>
માંડવડામાં મંગળગીતો ગવાય રે…
