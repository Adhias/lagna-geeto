# વરરાજે સીમડી ઘેરી

મોર તારી સોનાની ચાંચ, મોર તારી રૂપાની પાંખ</br>
સોનાની ચાંચે રે મોરલો મોતી ચરવા જાય

મોર જાજે ઊગમણે દેશ, મોર જાજે આથમણે દેશ</br>
વળતો જાજે વેવાયુંને માંડવે હો રાજ

વેવાઈ મારા સૂતા છે કે જાગ (2)</br>
અમારે વરરાજે સીમડી ઘેરી માણારાજ</br>
સીમડીએ કાંઈ ચમર ઢોળાવ (૨)</br>
ચમરનો હોંશી વીરો મારો આવ્યો માણારાજ

વેવાઈ મારા સૂતા છે કે જાગ (2)</br>
અમારે વરરાજે ઝાંપલા ઘેર્યા માણારાજ</br>
ઝાંપે કાંઈ છાંટણાં છંટાવ (૨)</br>
ઠંડકુંનો હોંશી વીરો મારો આવે માણારાજ

વેવાઈ મારા સૂતા છે કે જાગ (2)</br>
અમારે વરરાજે શેરીયું ઘેરી માણારાજ</br>
શેરીએ કાંઈ ફૂલડાં પથરાવ (૨)</br>
સુગંધીનો હોંશી વીરો મારો આવે માણારાજ

વેવાઈ મારા સૂતા છે કે જાગ (2)</br>
અમારે વરરાજે માંડવો ઘેર્યો માણારાજ</br>
માંડવડે કાંઈ લાડકી પધરાવ (૨)</br>
લાડકીનો હોંશી વીરો મારો આવે માણારાજ

ગોત્રીજે કાંઈ રમતું મંડાવ (૨)</br>
રમતુંનો હોંશી વીરો મારો આવે માણારાજ

મોર તારી સોનાની ચાંચ, મોર તારી રૂપાની પાંખ</br>
સોનાની ચાંચે રે મોરલો મોતી ચરવા જાય
