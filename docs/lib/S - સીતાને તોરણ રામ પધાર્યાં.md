# સીતાને તોરણ રામ પધાર્યાં

સીતાને તોરણ રામ પધાર્યાં</br>
લેજે પનોતી પહેલું પોંખણું

પોંખતાને વરની ભમર ફરકી</br>
આંખલડી રતને જડી

રવાઈએ વર પોંખો પનોતા</br>
રવાઈએ ગોરી સોહામણા

સીતાને તોરણ રામ પધાર્યાં</br>
લેજે પનોતી બીજું પોખણું

ધોંસરિયે વર પોંખો પનોતા</br>
ધોંસરિયે ગોરી સોહામણા

સીતાને તોરણ રામ પધાર્યાં</br>
લેજે પનોતી ત્રીજું પોખણું

ત્રાંકે વર પોંખો પનોતા</br>
ત્રાંકે રેટિયાં સોહામણા

સીતાને તોરણ રામ પધાર્યાં</br>
લેજે પનોતી ચોથું પોખણું

પીંડીએ વર પોંખો પનોતા</br>
પીંડીએ હાથ સોહામણા

વરરાજાને પોંખણ
