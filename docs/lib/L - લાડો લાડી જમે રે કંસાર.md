# લાડો લાડી જમે રે કંસાર

લાડો લાડી જમે રે કંસાર, કંસાર કેવો ગળ્યો લાગે રે</br>
લાડો લાડી જમે રે કંસાર, કંસાર કેવો ગળ્યો લાગે રે

નાખે મહીં ઘી કેરી ધાર, સંસાર પાયો ગળ્યો લાગે રે</br>
લાડો લાડી જમે રે કંસાર, કંસાર કેવો ગળ્યો લાગે રે

સાસુજી શુભ સજી શણગાર, પીરસવાને આવિયાં રે</br>
ભીની વડી સાકર તૈયાર, ઝારી ભરીને લાવિયાં રે

પીરસતાં મન મલકાય, આનંદ અંગ અંગમાં રે</br>
ભેગા બેસી જમે વરકન્યાય અધિક ઊંચા રંગમાં રે

પાસે બેઠી સૈયરો બે-ચાર તપાસ રાખે તે તણી રે</br>
રત્ને જડ્યો બાજોઠ વિશાળ મૂકે છે મુખ આગળ રે

લાડો લાડી જમે રે કંસાર, આનંદ આજ અતિઘણો રે</br>
લાડો લાડી જમે રે કંસાર, કંસાર કેવો ગળ્યો લાગે રે</br>
