# મારા નખના પરવાળા જેવી

મારા નખના પરવાળા જેવી ચૂંદડી</br>
મારી ચુંદડીનો રંગ રાતો હો લાડલી</br>
ઓઢોને સાહેબજાદી ચૂંદડી

હું તો કેમ કરી ઓઢું રે સાયબા ચૂંદડી</br>
મારા દાદાજી દેખે માતાજી દેખે</br>
કેમ રે ઓઢું રે સોરંગ ચૂંદડી

તમારા દાદાના તેડ્યાં અમે આવશું</br>
તમારી માતાના મન મોહશે હો લાડલી</br>
ઓઢોને લાડકવાયી ચૂંદડી

હું તો કેમ કરી ઓઢું રે સાયબા ચૂંદડી</br>
મારા વીરોજી દેખે ભાભી દેખે</br>
કેમ રે ઓઢું રે સોરંગ ચૂંદડી

તમારા વીરાના તેડ્યાં અમે આવશું</br>
તમારી ભાભીના ગુણલાં ગાશું હો લાડલી</br>
ઓઢોને સાહેબજાદી ચૂંદડી

ચુંદડી ઓઢણ
