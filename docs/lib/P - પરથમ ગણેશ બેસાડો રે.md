# પરથમ ગણેશ બેસાડો રે

પરથમ ગણેશ બેસાડો રે મારા ગણેશ દુંદાળા</br>
ગણેશ દુંદાળા ને મોટી ફાંદાળા</br>
ગણેશજી વરદાન દેજો રે હો… મારા ગણેશ દુંદાળા

ગણેશ દુંદાળા ને ફાંદે રૂપાળા</br>
તેત્રીસ કરોડ દેવતા સીમડીએ આવ્યા</br>
હરખ્યાં છે ગૌરીના મન રે હો મારા ગણેશ દુંદાળા</br>
હરખ્યાં છે માવડીના મન રે હો મારા ગણેશ દુંદાળા

ઊઠો ગણેશ ને ઊઠો પરમેશ</br>
તમે આવ્યા મારે રંગ રહેશે રે મારા ગણેશ દુંદાળા

વિવાહ ઘરણી ને જગન જનોઇ</br>
પરથમ ગણેશ બેસાડો રે મારા ગણેશ દુંદાળા
