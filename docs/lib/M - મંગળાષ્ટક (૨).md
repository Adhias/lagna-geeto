# મંગળાષ્ટક (૨)

દામ્પત્યે પગલાભરી પ્રણય નાં</br>
ધર્મે પ્રિતી રાખજો</br>
આશિષો પ્રભુ ની સદા વરસજો</br>
કુર્યાત સદા મંગલમ્

જે જે સ્વપ્ન તમે રચ્યા જીવન માં તે સહુ પ્રભુ પૂરજો</br>
રિદ્ધિ સિદ્ધિ અનેક્ય પ્રાપ્ત કરી ને</br>
કુર્યાત સદા મંગલમ્

અગ્નિદેવ ની સાક્ષી એ કર પ્રયાણ પૂરી પ્રતિજ્ઞા કરી...</br>
યાત્રા આ સંસાર ની શરૂ કરો</br>
કુર્યાત સદા મંગલમ્

કન્યા છે કુળ દીપિકા ગુણવતી..</br>
વિદ્યાવતી શ્રીમતી</br>
પહેરી ને પરિધાન મંગલ રૂડા</br>
કુર્યાત સદા મંગલમ્

કંઠે મંગળસુત્ર સુંદર દિસે</br>
મુકતાવલી ઉજ્વલમ્</br>
આપો એ પ્રિય કન્યકા સુખ ઘણું...</br>
કુર્યાત સદા મંગલમ્

ઉરમાં ઉમંગો લઇ અતિઘણા  પ્રભુતામાં પાડે પગલાં...</br>
એ પગલે સુખશાંતિ ને સમૃદ્ધિ</br>
કુર્યાત સદા મંગલમ્

સૌનીસાક્ષી એ સિંદૂર પૂર્યા સેથી એ જોડી અખંડ રાખજો</br>
અખંડ સૌભાગ્ય સદા રાખજો</br>
કુર્યાત સદા મંગલમ્

નવવધૂ ને, દીપાવજો કુળદેવી</br>
આશિષ દેજો અખંડ સૌભાગ્યનાં</br>
અમર.. રાખજો ચૂડી ને ચાંદલો</br>
કુર્યાત્ સદા મંગલમ્ (3)
