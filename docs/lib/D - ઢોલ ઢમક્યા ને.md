# ઢોલ ઢમક્યા ને

ઢોલ ઢમક્યા ને વરવહુના હાથ મળ્યા</br>
વાજા વાગ્યા ને વરવહુના હાથ મળ્યા</br>
હૈયાં હરખ્યા ને વરવહુના હાથ મળ્યા</br>
પ્રેમે નીરખ્યા ને વરવહુના હાથ મળ્યા</br>
જાણે ઈશ્વર ને પારવતી સાથ મળ્યા

જેમ નદી ને સરવરના જળ મળ્યા</br>
એમ વર ને કન્યાના હાથ મળ્યા</br>
જેમ દૂધમાં જાય સાકર ભળી</br>
એમ વર ને કન્યાની જોડ મળી</br>
જેમ શોભે લહેરો સાગરમાં</br>
એમ શોભે વર-કન્યા માયરામાં</br>
જેમ ઈન્દ્ર - ઈન્દ્રાણીની જોડ ઠરી</br>
એમ વર ને કન્યાની જોડ મળી

(હસ્તમેળાપ - વરપક્ષ)
