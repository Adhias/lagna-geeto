# નાણાવટી રે સાજન બેઠું માંડવે

નાણાવટી રે સાજન બેઠું માંડવે, </br>
લાખોપતિ રે સાજન બેઠું માંડવે

જેવા ભરી સભાના રાજા</br>

લાખોપતિ રે સાજન બેઠું માંડવે
