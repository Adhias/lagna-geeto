# ગણેશ પાટ બેસાડીએ

ગણેશ પાટ બેસાડીએ, ભલા નીપજે પકવાન</br>
સગા સંબંધી તેડીએ, જો પુજ્યાં હોય મોરાર…

જેને તે આંગણ પીપળો, તેનો તે ધન્ય અવતાર</br>
સાંજ સવારે પૂજીયે, જો પુજ્યાં હોય મોરાર…

જેને તે આંગણ ગાવડી, તેનો તે ધન્ય અવતાર</br>
સાંજ સવારે દોહી વળે, જો પુજ્યાં હોય મોરાર…

જેને તે આંગણ દિકરી, તેનો તે ધન્ય અવતાર</br>
સાચું સુરયુ ધન વાપરે, જો પુજ્યાં હોય મોરાર…

જેને તે પેટે દીકરી, તેનો તે ધન્ય અવતાર</br>
વહુ વારુ પાયે પડે, જો પુજ્યાં હોય મોરાર…
